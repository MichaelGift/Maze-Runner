    About

    This project was inspired by my love for gaming, game design, and the C language.
    I wanted to take players on a journey to the past, capturing the essence of classic games both visually and experientially.
    By using SDL (Simple DirectMedia Layer) and C, I aimed to create a game that harkens back to the golden era of gaming, where simplicity and gameplay reigned supreme.
